## libfuse
1. 进入 `libfuse` 目录后，先完整清理旧安装（避免混用旧库）：
```bash
sudo rm -f /usr/local/lib/libfuse3*
sudo rm -f /usr/local/lib/x86_64-linux-gnu/libfuse3*
sudo rm -rf /usr/local/include/fuse3
sudo ldconfig
```
2. 清理本地构建目录并重新构建：
```bash
rm -rf build
meson setup build
ninja -C build
```
3. 安装并刷新动态库缓存：
```bash
sudo ninja -C build install
sudo ldconfig
```
4. 安装后校验（必须确认一致）：
```bash
pkg-config --modversion fuse3
pkg-config --cflags --libs fuse3
ldconfig -p | grep libfuse3.so.4
```
