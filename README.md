## 四、项目目标与规划

* [ ] 完成 eBPF Hook 原型
* [ ] 实现高效的用户态缓存机制
* [ ] 支持多核并发调度与 NUMA 优化
* [ ] 提供完整的 CLI 接口与挂载管理
* [ ] 完善测试与性能评估框架
