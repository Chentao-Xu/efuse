## 快速开始

### 环境依赖

* Linux 5.10+ 内核
* `clang`, `llvm`, `libbpf-dev`
* 支持 FUSE3 的操作系统
* 可选：Docker / Make

### 构建方法

```bash
git clone https://github.com/your-org/efuse.git
cd efuse
make
```
